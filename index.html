How I managed this site

To be honest, creating a dedicated site from scratch with HTML isn't that easy as people imagined. But if you got a hobby to write codes, welp, I can't judge you. üóø

For make things easier, I'm using GitHub Pages for deploying this site, Font Awesome for embedding icons, jQuery for JavaScript library, and W3.CSS for CSS framework (so I won't have any headache while designing the site lol).

Also, if you look at the source code of this site, you will found that somehow, I can manage over ~398 cards bruh. Big thanks to clubs index (in TOML) and card editions index (in JSON) I've made to assist me on placing the cards in their place.

Fun fact: there's a schema file for edition index JSON. try to look it at editionDetail.schema.json.
